# juejin-vue
> 仿掘金网站

## 技术栈

> 暂定就是有可能用，但是目前还没用的。

+ 语言：JavaScript
+ 框架:  Vue.js，Nuxt.js
+ UI风格：Bootstrap
+ 包管理：npm
+ 接口请求： Axios
+ 状态管理：Vuex
+ 测试工具：Jasmine
+ 提交规范：commitlint，husky

## 项目运行
```shell
# 安装依赖
npm install

# 运行项目
npm run dev

```
